## NAME
Generate a JSON Schema compatible with this version of Karsk

## SYNOPSYS
**karsk schema** **--output file**

## DESCRIPTION

Uses Pydantic's built-in JSON Schema generation to create a schema for use in other tools, notably in [YAML Language Server](https://github.com/redhat-developer/yaml-language-server).

A casual user need only to have the following preamble at the top of a Karsk configuration YAML file

```yaml
# yaml-language-server: $schema=https://equinor.github.io/karsk/schema.json
```

To use the schema generated by this command directly, change the path after `$schema=` to point to your new schema file

## OPTIONS

#### **--output** *file*
Write JSON Schema to given *file*. Defaults to writing the schema to STDOUT.
