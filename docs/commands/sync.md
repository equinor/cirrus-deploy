# karsk sync

!!! warning
    This section describes a planned feature. Actual behaviour in Karsk may differ.

## NAME
karsk\-sync - Synchronise a Karsk environment

## SYNOPSIS
**karsk sync** *config* *areas*

## DESCRIPTION
**karsk sync** copies data from *destination* as specified in the *config* on localhost to *destination* on all the areas described in the *areas* file.
