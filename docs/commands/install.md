# karsk install

!!! warning
    This section describes a planned feature. Actual behaviour in Karsk may differ.

## NAME
karsk\-install - Install a Karsk environment

## SYNOPSIS
**karsk install** *config*

## DESCRIPTION
**karsk install** copies the build data from the staging directory to the *destination* location as defined in the *config* file, as well as the wrapper *runscript*.
